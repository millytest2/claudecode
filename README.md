# Creator OS (Local + Free)

A browser-based content repurposing and platform-intelligence workspace that turns one core idea into platform-ready outputs while preserving your voice.

## Why this exists

- No API cost (runs locally in browser)
- Fast repeatable outputs for multiple platforms
- Profile and research memory (saved in localStorage)
- Platform-by-platform strategic buckets so you can keep up with what works now

## Run

Open `index.html` directly, or use a local static server:

```bash
python3 -m http.server 4173
```

Then visit `http://localhost:4173`.

## Current capabilities

- Voice profile (brand, audience, tone, signature phrases)
- Research Agent Inputs:
  - research window (7/14/30 days)
  - niche keywords
  - accounts to monitor
  - winning post samples
- Platform strategy buckets per selected platform:
  - what is working now
  - hook types
  - CTA patterns
  - content pillar ideas
  - research sprint checklist + search tactics
- Draft outputs for:
  - X / Threads
  - LinkedIn
  - Instagram Caption
  - TikTok Script
  - YouTube Short
  - Email
- Multiple variations per platform
- Copy all output and export to `.txt`
